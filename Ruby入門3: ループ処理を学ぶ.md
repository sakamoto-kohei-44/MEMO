****01:0から9までを表示してみよう - for in 受講済みにする****

```jsx
for カウンタ変数 in 繰り返す範囲 do # doは省略可能
繰り返し処理
end

for i in 0..9
    puts "hello"
end
```

***02:条件によるくり返し処理1 - while***

```jsx
for カウンタ変数 in 繰り返す範囲 do # doは省略可能
繰り返し処理
end

for i in 0..9
    puts "hello"
end
```

# ****03:条件によるくり返し処理2 - while 受講済みにする****

```jsx
# whileによるループ処理

i = 5	# カウンタ変数を初期化
while i <= 15
    puts i		# 繰り返し処理
    i = i + 2	# カウンタ変数を更新
end
puts "last #{i}"
```
